# NYC Airbnb Data Analysis
## Overview
This project is my "introduction" to the world of data science. It includes a beginner-level exploratory data analysis (EDA) using Python and Airbnb listing data from New York City.

The goal of this project was to practice:
* Data cleaning
* Visualization
* Correlation analysis
* Utilization of real-wrold datasets

## Dataset
Dataset source: [New York City Airbnb Open Data](https://www.kaggle.com/datasets/dgomonov/new-york-city-airbnb-open-data?utm_source=chatgpt.com)\
I used the linked dataset to explore trends in pricing, neighborhoods, reviews, host activity, and listing availability. The dataset contains various information about Airbnb listings in the city, including:
* Host information
* Listing description
* Location
* Price
* Reviews

## Tools & Libraries

## Project Goals
Once again, some of the questions I explored in this project include:
* What are interesting features of the dataset?
* Which neighborhoods and neighborhood groups, otherwise known as boroughs, have the highest and lowest average Airbnb prices?
* What is the relationship between price and the other features?
* Do listings with higher availability receive more reviews?
* What words commonly appear in Airbnb listings?
* What characteristics define "busy" hosts?
